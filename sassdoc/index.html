    align

    @mixin align($x, $y) { ... }

    Description

    Align items using the transform element

    Parameters

    parameterNameparameterDescriptionparameterTypeparameterDefault value
    $x

    Set the horizontal alignment, left, center, right or none. none returns inherit

    		String—none
    $y

    Set the vertical alignment, top, center, bottom or none. none returns inherit

    		String—none

    Example

    .xAlign {
  @include align('center', 'none');
}

    Used by

    • [mixin] logo

    Author

    • Bob Wassermann

    logo

    @mixin logo($position) { ... }

    Description

    Add the logo in the default size. Uses the variables $logo--width and $logo--height, defined in _helpers.scss

    Parameters

    parameterNameparameterDescriptionparameterTypeparameterDefault value
    $position

    can be set to absolute, which will align the icon on the y-axis too. Parameter can be left open to just get teh logo without extra styling

    		String—none

    Example

    .logo {
  @include logo();
}

    • [mixin] align
